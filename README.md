# DigyDukaan

A stunning, minimalistic e-commerce template built for efficiency and simplicity.

## Tech Stack
- **React 19**
- **Vite**
- **Tailwind CSS**
- **Lucide React** (Icons)
- **Framer Motion** (Subtle Animations)

## Getting Started

1. **Install Dependencies**:
   ```bash
   npm install
   ```

2. **Run Development Server**:
   ```bash
   npm run dev
   ```

3. **Build for Production**:
   ```bash
   npm run build
   ```

## Design Principles
- **Clarity**: Use of high-contrast typography and generous whitespace.
- **Minimalism**: Focus on the product, not the chrome.
- **Trust**: Clean, professional layout similar to modern direct-to-consumer brands.
